# Drishtee Computer Institute Website

**Empowering Rural India Since 2007**  

Welcome to the official website of **Drishtee Computer Institute**, the premier destination for computer education in Eastern Uttar Pradesh. The institute has been transforming lives through digital literacy and professional IT training for over 19 years.

## About the Project

This project is a **React + Vite frontend application** for Drishtee Computer Institute. It includes:

- Public pages like Home, About, Courses, Gallery, Branches, Library, and Contact forms.  
- Student and Admin dashboards with role-based access control.  
- Firebase authentication and role management (Admin/Student).  
- Offline capabilities via an install prompt (PWA ready).  
- Network status detection and graceful handling for offline scenarios.  
- Dynamic SEO management using a `HelmetManager` component.  

The website is mobile-first and responsive, designed to deliver a modern, user-friendly experience to both students and staff.

---

## Features

### Public Features
- Home, About, Courses, Branches, Gallery, Library, and Offers.  
- Admission and query forms.  
- Certificate verification and download functionality.  
- Interactive course pages for Computer Languages, Web Development, Designing, Banking, NIELET, and more.  
- Google Maps integration for branch locations.  

### Protected Features
- **Admin Dashboard:** Manage students, courses, and admissions.  
- **Student Dashboard:** Access personalized content, course materials, and certificates.  
- Role-based routing to ensure proper access control.

### Other Features
- Install prompt for PWA installation.  
- Network status alert when offline.  
- 404 Page for invalid routes.  
- Modern, responsive UI with dynamic content management.

---

## Tech Stack

- **Frontend:** React.js, Vite  
- **Routing:** React Router DOM  
- **Authentication & Database:** Firebase Auth & Firestore  
- **Styling:** Custom CSS framework (`hbstyle.css`) and responsive layouts  
- **Utilities:** React Toastify (notifications), html2pdf.js (certificate download)  
- **SEO:** React Helmet (via `HelmetManager`)  
- **Progressive Web App (PWA) Features:** Install prompt, offline network handling
